# 🎉 Personal Introduction Program

A clean, beginner-friendly Python project that collects user details and displays a personalized welcome message.

---

## 📌 Project Overview

This project demonstrates core Python fundamentals by building a simple interactive CLI program. It asks the user for basic information and prints a friendly, formatted greeting.

---

## 🎯 Objectives

* Collect user input using `input()`
* Store values in variables
* Format output using **f-strings**
* Display results using `print()`
* Practice basic program flow

---

## 🛠️ Tech Stack

* **Language:** Python 3
* **Environment:** CLI / Terminal / VS Code / IDLE

## ⚙️ Setup & Run

### 1. Install Python

Download from: [https://www.python.org](https://www.python.org)

### 2. Clone Repository

```bash
git clone https://github.com/ShreyaDesai23/personal-introduction-program.git
cd personal-introduction-program
```

### 3. Run Program

```bash
python personal_intro.py
```

---

## ▶️ How It Works

1. Prompts the user for:

   * Name
   * Age
   * Hobby
   * City
2. Stores responses in variables
3. Prints a formatted welcome message
